Description:

Plugin Masmons WP Shares merupakan plugin Wordpress gratis yang dapat Anda gunakan untuk menghadirkan fitur share post ke sejumlah social media.

Terdapat sejumlah opsi yang menarik dan bisa membuat website Anda berbeda dengan website lainnya.

==Fitur Masmons WP Shares==
- pilihan desain (template, layout style1, style2, style3 dst)
- pilihan trigger
- floating trigger
- show share offer container after xx% screen height
- shortcode to insert in your content
- disable share in specific post by ID or all posts in specific category(s) or all posts in specific tag(s)
- customable share position
- customable share container background
- use your own icon (in .svg format)


==Dowload dan Instalasi==
Plugin Masmons WP Shares membutuhkan jQuery dan plugin Carbon Fields min versi 3.6.0 harus sudah terinstall dan aktif didalam website Anda. Silahkan download, install dan aktifkan plugin Carbon Fields (100% gratis, legit, secure).

Dowload Carbon Fields: https://carbonfields.net/release-archive/
Support: https://budiharyono.com/ - https://www.facebook.com/groups/946801826538446


--Instalasi--
- Setelah plugin Carbon Fields aktif
- Download, install dan aktifkan plugin Masmons WP Shares
- Konfigurasi
- Selesai
